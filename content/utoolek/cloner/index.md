A minimal CLI tool for cloning git repositories.

- Supports **multiple URL formats**
- Works with GitHub, GitLab, Gitea

<!--more-->

## Installation

```bash
pip install cloner
```

## Usage

```bash
cloner https://github.com/user/repo
```
