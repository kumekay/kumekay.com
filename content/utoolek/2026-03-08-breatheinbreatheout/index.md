Some sites can be addictive. Blocking them completely can be a solution, but sometimes I still want to visit them, just I don't want to do it mindlessly. This is a userscript that adds 15-second timer before accessing distracting websites.

The timer resets if you switch tabs, so you must wait in foreground.

Inspired by [Mindful Internet Use](https://github.com/IvarssonAndreas/mindful-internet-use) which doesn't work in Chrome anymore. And Tampermonkey (or similar userscript extension) can be used on mobile: with Safari on iOS and Firefox on Android.

<!--more-->

## Installation

1. Install the [Tampermonkey browser extension](https://www.tampermonkey.net/)
2. Install from [Greasy Fork](https://greasyfork.org/en/scripts/564140-domain-access-timer)

Or install directly from GitHub: [breatheinbreatheout.user.js](https://github.com/kumekay/breatheinbreatheout/raw/refs/heads/main/breatheinbreatheout.user.js)

## Default Domains

- reddit.com
- facebook.com
- x.com
- youtube.com

YouTube Music (`music.youtube.com`) is excluded by default.
